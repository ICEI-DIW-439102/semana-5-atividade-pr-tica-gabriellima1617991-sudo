Sobre o Sistema

O sistema LPC Restaurantes foi desenvolvido com o objetivo de recomendar estabelecimentos gastronômicos com base na localização do usuário. A plataforma busca oferecer uma experiência moderna e intuitiva, permitindo que os usuários encontrem opções de alimentação de forma rápida e eficiente.

O sistema apresenta restaurantes fictícios com base em categorias variadas, como culinária brasileira, italiana e japonesa. Cada restaurante possui informações básicas, além de opções de visualização de detalhes e envio de feedback.

A interface foi projetada com foco em usabilidade, responsividade e organização visual, utilizando conceitos modernos de design, como cards, cores harmônicas e layout adaptável para diferentes dispositivos.

Este projeto foi desenvolvido como atividade acadêmica, com o objetivo de aplicar conceitos de HTML, CSS e JavaScript na construção de uma aplicação web funcional e visualmente agradável.
